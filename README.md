# 📱 MO:BILE

<br>

**가상 모바일 기기 쇼핑몰**을 제작<br>
- 제품 및 요금제를 비교하여 간편하게 구매할 수 있는 모바일 기기 쇼핑몰을 기획
- 회원 관리, 제품 관리, 게시판을 CRUD를 통해 구현
- PC와 모바일 디바이스에 대응할 수 있도록 반응형으로 제작

<br>
